# Architecture Overview

> Last updated: 2025-12-22
> Generated by: map-codebase skill

## System Purpose

A **license management system** for software products with a reseller distribution model. Internal admins manage products and resellers; resellers manage their allocated licenses and end-users.

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | Next.js 16.1.0 (App Router) |
| Language | TypeScript 5 |
| Runtime | React 19 |
| Database | PostgreSQL via Drizzle ORM |
| Auth | NextAuth v5 (Credentials provider, JWT sessions) |
| State/Cache | TanStack React Query |
| Styling | Tailwind CSS v4 |
| UI Components | shadcn/ui (Radix primitives) |
| Forms | React Hook Form + Zod validation |
| Logging | Pino |
| Icons | Tabler Icons |

## Build & Deploy

```bash
# Development
npm run dev          # Start Next.js dev server

# Database
npm run db:push      # Push schema to DB
npm run db:generate  # Generate migrations
npm run db:migrate   # Run migrations
npm run db:studio    # Open Drizzle Studio

# Production
npm run build        # Build for production
npm run start        # Start production server
npm run lint         # Run ESLint
```
